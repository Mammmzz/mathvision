MathVision adalah alat bantu visualisasi matematika berbasis web yang dirancang untuk membantu siswa dan pengajar memahami konsep persamaan dan transformasi grafik secara mudah dan interaktif.

---

## âœ¨ Fitur Utama

- **Visualisasi Real-time**: Ubah persamaan matematika menjadi grafik secara instan.
- **Transformasi Interaktif**: Lakukan refleksi, dilatasi, dan translasi pada grafik dengan sekali klik.
- **Mode Perbandingan**: Bandingkan grafik asli dengan versi transformasinya menggunakan slider interaktif.
- **Inspeksi Titik**: Cari tahu nilai `y` dari titik `x` mana pun pada grafik.
- **Mode Aksesibilitas**: Dapatkan deskripsi teks lengkap dari sebuah grafik.
- **Simpan Grafik**: Ekspor visualisasi Anda sebagai gambar PNG untuk bahan ajar atau catatan.
